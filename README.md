# War Logistics Commander

This is my logistics simulation game project.

The project has 5 submissions:

1. LogSim 1 — Basic supply mission
2. LogSim 2 — Multiple frontline priority
3. LogSim 3 — Forecast and crisis
4. LogSim 4 — Upgrade and performance
5. LogSim 5 — Final connected game

LogSim 1 to LogSim 4 are development prototypes.

The final game is `index.html`.

In the final game, the player moves through 5 connected logistics stages:

1. Procurement
2. Storage
3. Transportation Planning
4. Transportation / Distribution
5. Feedback / Evaluation

The player carries information from one stage to the next:

- budget
- food
- fuel
- ammo
- medical supplies
- morale
- base health
- convoy health
- frontline condition
- XP
- upgrades

Main idea:

War is not only fighting. War is logistics.
